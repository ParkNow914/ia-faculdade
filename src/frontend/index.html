Sobre o Projeto

🎯 Objetivo

Sistema de previsão de consumo de energia elétrica utilizando redes neurais LSTM (Long Short-Term Memory) para auxiliar na gestão eficiente de recursos energéticos.

🛠️ Tecnologias

  • TensorFlow/Keras (Deep Learning)
  • FastAPI (Backend API)
  • Python (IA e Backend)
  • HTML/CSS/JS (Frontend)
  • Chart.js (Visualizações)

📊 Dataset

Dados sintéticos baseados em padrões reais de consumo de energia, incluindo variáveis como temperatura, hora do dia, dia da semana, e padrões sazonais.

🎓 Projeto Acadêmico

Desenvolvido para a disciplina de IA do curso de Gestão de T.I., demonstrando aplicações práticas de Deep Learning em problemas reais de gestão de recursos.
